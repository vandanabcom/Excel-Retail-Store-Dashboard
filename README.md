# Excel-Retail-Store-Dashboard
📌 Project Overview

This project demonstrates how Microsoft Excel can be used to analyze retail sales data and create an interactive business dashboard.

The objective of this project is to monitor sales performance, profit, customer trends, and product performance, enabling management to make better business decisions.

🎯 Project Objective
Clean and prepare sales data
Analyze sales performance
Build an interactive dashboard
Generate business insights

🛠 Tools Used
Microsoft Excel

📚 Excel Skills Used

Data Cleaning
XLOOKUP
IF & IFERROR
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Dashboard Design

📂 Dataset

The project uses a synthetic retail sales dataset created for learning and portfolio purposes.

Dataset Includes
Customer Details
Product Information
Sales Transactions
Revenue
Profit
Quantity
City
Category

📊 Dashboard KPIs

Total Revenue
Total Profit

Total Orders

📈 Dashboard Features

Revenue by Product
Revenue by City
Profit by Category
Monthly Revenue Trend
Monthly Profit Trend
Top 10 Customers
Interactive Filters (Slicers)

💡 Key Business Insights

Laptop generated the highest revenue.

Jaipur was the top-performing city.

Electronics contributed the highest share of revenue.

Revenue and profit peaked in January, March, and December.

The dashboard helps identify high-performing products and regions.
